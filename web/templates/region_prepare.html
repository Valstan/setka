Загружаю настройки региона…

Зачем этот шаг? VK-поиск выдаёт мусор, если просто искать по названию района — выскакивают крупные областные паблики. Скажи программе, какие населённые пункты входят в район и какие тематические слова стоит искать — тогда discovery отфильтрует нерелевантных кандидатов автоматически, без AI.
1. Населённые пункты района

Список городов, ПГТ, сёл и деревень района. Используется как жёсткий фильтр: кандидат без хотя бы одного локалитета в названии или описании отбрасывается. Можно подгрузить автоматически из OpenStreetMap, либо скопировать из ChatGPT по готовому prompt'у ниже, либо вписать руками.

Название района в OSM
Альтернатива: скопировать готовый prompt для ChatGPT/Claude
2. Тематические ключевики (опц.)

Слова, по которым VK будет искать тематические паблики района. Если оставить пусто — программа использует стандартный набор (новости, объявления, ДТП, культура, спорт и т. п.). Кастомные ключевики имеют смысл для районов со специфическими словами («вятский», «тужанский») или уникальными локальными терминами.

Готовый prompt для ChatGPT/Claude
Запустить discovery

Программа выполнит VK-поиск по нп и ключевикам, отфильтрует нерелевантных кандидатов по корню топонима, отсортирует по числу совпадений + подписчики, и откроет страницу проверки. Может занять от 30 секунд до 2 минут.
