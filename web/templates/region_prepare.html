Подготовка поиска сообществ — {{ region_code }}

Загружаю настройки региона…

Зачем этот шаг? VK-поиск выдаёт мусор, если просто искать по названию района — выскакивают крупные областные паблики. Скажи программе центр района, какие населённые пункты входят в район и какие тематические слова стоит искать — тогда поиск отфильтрует нерелевантных кандидатов автоматически.
0. Базовые настройки района обязательно

Центральный город района (например, «Тужа», «Малмыж», «Верхошижемье») — используется как корневой стем для фильтра релевантности и префикс для VK-поиска. Без него поиск сообществ не запустится. У старых регионов это поле могло остаться пустым — заполни сейчас.

1. Населённые пункты района

Список городов, ПГТ, сёл и деревень района. Используется как жёсткий фильтр: кандидат без хотя бы одного локалитета в названии или описании отбрасывается. Скопируй готовый prompt ниже в ChatGPT/Claude и вставь ответ в поле «Список нп», либо вписывай руками.

2. Тематические ключевики (опц.)

Слова, по которым VK будет искать тематические паблики района. Если оставить пусто — программа использует стандартный набор (новости, объявления, ДТП, культура, спорт и т. п.). Кастомные ключевики имеют смысл для районов со специфическими словами («вятский», «тужанский») или уникальными локальными терминами.

Запустить поиск сообществ

Программа выполнит VK-поиск по нп и ключевикам, отфильтрует нерелевантных кандидатов по корню топонима, отсортирует по числу совпадений + подписчики, и откроет страницу проверки. Запрос уходит в фон через Celery — UI не зависает. Для крупных районов (50+ нп) полный поиск занимает 5-10 минут.
