# Офферные картинки рекламного кабинета

Положите сюда картинки с вашим предложением о размещении рекламы (прайс,
портфолио, условия) — `*.jpg` / `*.jpeg` / `*.png`. При отправке ответа автору
предложки из `/ad-cabinet` они автоматически прикрепятся к личному сообщению
(до 5 штук, в алфавитном порядке имён файлов).

⚠️ Картинки уходят в ЛС **community-токеном** отправляющего сообщества — если у
группы нет community-токена в `/tokens`, оффер уйдёт текстом без картинок.

Файлы картинок в git намеренно не коммитятся (см. `.gitignore` рядом) — это
контент владельца, а не код.
