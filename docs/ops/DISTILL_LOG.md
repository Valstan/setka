# Журнал дистилляций Корпуса

> **Корпус** — накапливаемый обучающий материал классификатора: посты + вердикты
> нейросети + правки оператора (`content_classifications`, хранится вечно).
> **Дистилляция** — разбор корпуса и чеканка обобщённых правил для
> файла-корректировщика; делается из dev-чата скиллом [`/distill`](../../.claude/commands/distill.md)
> (облачная рутина-дистиллятор удалена владельцем 2026-07-14).
>
> Свежее сверху. По верхней дате `/start` напоминает, если прошло > 7 дней.
