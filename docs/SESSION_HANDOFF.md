## Failed approaches (этой нитки)

- **Живое веб-утверждение правил через git-файл** — отвергнуто в дизайне: web-процесс не может/не должен
  git-коммитить на каждый клик (нарушает PR-only ADR-0002, фрагильно). Решение — БД-overlay + периодический
  снапшот в git (ADR-0005). Не возвращаться к «правила только в файле» для живой петли.
- **Промпт рутины без контекста авторизации** — ИИ рутины принимал задачу за модерацию чужой платформы и
  отказывался. Лечение — не обход, а правдивый контекст (чей сервис, что делают вердикты, shadow+HITL) +
  разведение egress-блока и этики. Зафиксировано в `docs/ops/hitl-classifier-routine.md` §Troubleshooting.
