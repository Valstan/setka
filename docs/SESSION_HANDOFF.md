  (VK error 5) стоял первым в single-pick чтении без fallback → 0 постов → 0 сводок.

  VALSTAN (env+БД+рестарт) вернули сводки в эфир.
- **#336** — БД единый источник токенов (env только bootstrap); каскад публикации
  community → VALSTAN → резерв; самолечащееся чтение (probe `users.get` → cooldown →
  следующий токен). Все env-хардкоды `VK_TOKENS.get("VALSTAN")` переведены на БД-роутер.
- **#337** — карусель чтения: READ-токены по очереди (`last_used ASC` + штамп при
  выборе), нагрузка парсинга равномерно по VALSTAN+VITA+МАМА.

  Каскад `community → VALSTAN → МАМА` рабочий во всех эшелонах; Vita — read-only.
