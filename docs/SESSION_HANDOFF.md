## Failed approaches (этой нитки)

- **smoke-тест через web-эндпоинт diagnostics** — POST `/api/regions/{code}/diagnostics` закрыт
  session-гейтом `AuthGateMiddleware`, у post-deploy скрипта нет cookie → 401. **Не возвращаться
  к HTTP-пути** — правильно ставить задачу напрямую в Celery (внутренний seam без гейта, #329).
- **Промпт-путь для over-refusal облачной рутины** — исчерпан ещё в прошлой сессии (v1/v2/нейтральный
  все отбиты). Дистиллятор сразу построен на v3 (ключ+HTTP в код), чтобы не наступить на те же грабли.
