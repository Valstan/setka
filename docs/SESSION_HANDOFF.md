- **`krugozor` в пуле `arbazh` — не трогать.** 12 сообществ выглядят как «неканонная категория
  под чистку», но их выбирает `execute_krugozor_broadcast` **без фильтра по региону**; уборка

  «сбор встал». Правильно — `> (now() at time zone 'UTC') - interval 'N'`.

## Failed approaches (этой нитки)

- **Второй проход БЕЗ `--region-filter`** — сортировка по подписчикам без locality-скоринга
  вытаскивает федеральных гигантов («ДЕТСКИЙ САД — вопросы и ответы», 79k) вместо районных
  садиков, а мелкие районные группы обрезаются `--top`. Обязательны `--region-filter` по корню
  прилагательного + `--per-label-top`.
- **`args` воркфлоу массивом** — приезжает **строкой**, `Array.isArray` возвращает false, веер из
  N агентов молча схлопывается в одного (воркфлоу при этом не падает). Парсить: `typeof v === 'string'
  → JSON.parse`. Проверять `log()`-ом длину списка сразу после старта.
- **Мерить окна через `now() - interval`** — см. находку про UTC/MSK выше.
- **`pkill` через `ssh` с `&&`-цепочкой** — SSH на этот хост периодически отдаёт exit 255
  (`Connection timed out during banner exchange` / `Connection reset`); это флап канала, а не отказ
  команды. Ретраить с бо́льшим `ConnectTimeout`, не диагностировать как ошибку скрипта.
