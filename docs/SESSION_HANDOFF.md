## Failed approaches (этой нитки)

- **`curl` к `/api/classifier-review/health` с прода** — отдаёт `{"detail":"Not authenticated"}`
  (эндпоинт за AuthGate, сессионная кука). Для замеров ходить **напрямую в psql**, а не через
  этот эндпоинт; он для браузера оператора.
- **`propose` с длинным `rule_text`** — сервер валидирует 3..600 символов и режет весь батч
  (`rule_text не строка 3..600 символов`). Держать формулировку правила под 600.
