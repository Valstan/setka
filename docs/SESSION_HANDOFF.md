## Failed approaches (этой нитки)

- **Полностью включить `is_advertisement` для темы reklama** — отвергнуто: убило бы всю рекламную рубрику
  (в ней каждый пост — реклама, это доска объявлений). Решение — **узкий** `is_hard_spam` поверх байпаса:
  режет только скам/увод-в-обход-VK, легальные объявления пропускает. Не возвращаться к «фильтровать рекламу
  в рекламной рубрике целиком».
- **Считать, что публикация через community-токен снимает риск бана** — неверно: VK банит аккаунт-админа за
  КОНТЕНТ независимо от токена. «Другой токен без привязки к аккаунту» физически не существует. Лечится
  контентом, не маршрутизацией токенов.

## Открытые вопросы для пользователя

- **is_advertisement + is_hard_spam глобальны** (все районы). Аудит обеих сторон включён только для `mi` →
  over-drop в других районах не виден напрямую. Если понадобится per-region — откат тривиален.
- Заводить ли community-токены для остальных целевых пабликов (чтобы публикация никогда не откатывалась на
  личный VALSTAN)? Не блокер — но снимает *прямую* ответственность аккаунта. Главный рычаг всё равно контент.
