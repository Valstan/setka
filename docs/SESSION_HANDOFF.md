## Failed approaches (этой нитки)

- **Telegram-бот → `@smartspeech_sber_bot` для распознавания** — невозможно: Bot API запрещает
  боту писать боту. Автоматизация только через userbot (Telethon, личный аккаунт) = серая зона
  ToS / риск бана. Владелец отказался. **Не возвращаться к bot-route.**
- **«Веб-страница из Сарафана сама ставит и гоняет нативный код на любом компе»** — невозможно:
  песочница браузера запрещает сайту установку/исполнение нативного кода на клиенте. Реальный
  локальный compute через браузер = либо localhost-помощник (ставится 1×), либо WASM в самой
  вкладке. **Не обещать «автоустановку с любого компа».** (Итог: аудио-модуль и ушёл к Мозгу как
  локальный десктоп-скил — это и есть «localhost-помощник, ставится 1×».)
