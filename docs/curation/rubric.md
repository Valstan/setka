# Рубрика релевантности для LLM-курации сводок

> Растущая рубрика (а не дообучение модели) — носитель «практического обучения»
> фильтра. Ревьюишь ошибки → дописываешь правило/пример здесь → следующий прогон
> `/curate` умнее. Письмо brain 2026-06-07 «LLM-курация сводок» §«Обучение — честно».

Каждый кандидат в `digest_curation_runs.candidates` **уже прошёл** все
детерминированные фильтры (реклама/религия/дедуп/relevance) и был опубликован.
Задача LLM — поймать то, что алгоритм пропустил. Любой `drop` = чистая дельта
над текущим алгоритмом, поэтому **default = keep**: ставь `drop` только при
уверенности, иначе мы плодим ложные срабатывания (это и есть метрика precision,
которую мерим в Фазе 1).

## Вердикт `drop`, если пост попадает в один из классов

1. **Реклама / коммерция**, проскочившая ad-фильтр: продажа товара/услуги, набор
   на курсы, «пишите в директ», прайс, промокод, агрессивный призыв купить.
   *Не* реклама: анонс бесплатного районного мероприятия, вакансия от
   госучреждения, соцобъявление.
2. **Нерелевантно району**: событие/тема без привязки к данному району
   (общегородской/федеральный контент, омонимичный топоним другого региона,
   развлекательный репост без локального повода). Привязка = упомянут
   район/его населённый пункт/местное учреждение, **или** очевидно местное
   событие.
   **Исключение — познавательный/научно-популярный/общеобразовательный контент**
   (наука, технологии, история-факт, природа, медицинский ликбез): публикуется
   **намеренно** для расширения кругозора жителей **даже без местной привязки** →
   `keep`. Решение владельца 2026-06-14: «новости науки должны публиковаться, у
   людей расширяется кругозор». Граница vs «развлекательный репост»: познавательное
   несёт **факт/знание** (как устроено, что открыли, что произошло в истории),
   развлекательный — пустой репост ради лайков («первые маслята», «день друзей» со
   стоковым фото). Сомнение между ними → `keep` (default = keep).
3. **Переписанный дубль**: смысловой повтор другого поста этого же прогона
   (перефраз, та же новость другими словами) — simhash такое не ловит. Оставляй
   первый/полнейший вариант (`keep`), помечай `drop` повторам с reason
   «перефраз-дубль #<lip первого>».

## Вердикт `keep` (оставить) — всё остальное

Нормальная районная новость, культурное/спортивное событие, объявление
госоргана, локальное происшествие, поздравление, объявление о пропаже и т.п.
**Познавательное/научно-популярное/общеобразовательное** (наука, история-факт,
технологии, природа, ликбез) — `keep` даже без местной привязки (см. drop-класс
§2 исключение): редакционная политика — расширять кругозор жителей.

## Принцип reroute, а не drop (для enforcing-дизайна Фазы 2)

Решение владельца 2026-06-14: реклама, попавшая **не в свою тему** (напр. продажа
в `novost`), в идеале должна **переезжать в блок `reklama`**, а не просто
выкидываться. На Фазе 1 (shadow) LLM по-прежнему ставит `drop` (мы только мерим), но
при проектировании enforcing закладываем **переадресацию off-theme рекламы в
`reklama`**, а не безусловное удаление. Полное удаление — только для спама/мусора без
тематической принадлежности вообще.

## reason — кратко и машинно-группируемо

Короткая каноничная причина (она агрегируется в `--stats` → `drop_reasons`):
`реклама`, `не-район`, `перефраз-дубль`, `федеральный`, `развлекательный-репост`.
Можно дописать `: деталь`.

## Per-region заметки (дополняется по ходу PoC)

### <region_code> (название)

- _Накопленные особенности района: типичные омонимы, локальные паблики, частые
  ложные срабатывания. Дописывай сюда после ревью ошибок._

### mi (Малмыжский район)

- **Паблик `216215303`** постит ежедневные исторические заметки «N июня YYYY г. …»
  («в этот день в истории») — общероссийский патриотический контент без привязки
  к району. → `drop` `не-район: историческая дата без местной привязки`.
- **Областной (Кировская обл.) контент — основная серая зона.** Эвристика после
  PoC-прогона 2026-06-08:
  - `drop` (`не-район`): генеричные областные сводки/промо без местного хука —
    геомагнитные бури, всероссийские конкурсы, областные квест-туры с розыгрышем.
  - `keep`: областное с **практической пользой для жителя или призывом к действию** —
    предупреждение о непогоде с горячей линией энергетиков, голосование ФКГС
    (благоустройство, среди объектов есть местные), региональная прокуратура о
    мошенничестве. Граница: «могу ли я (житель Малмыжа) что-то с этим сделать
    прямо сейчас?» → да = keep.
- **Барахолка `24611937`**: продажи с ценой/телефоном — `реклама` **только если
  пост утёк в НЕ-рекламную тему** (`novost` и т.п.) → `drop`. Если тема-цель прогона
  = `reklama` (сводка «Объявления»), продажи **штатны** → `keep` (см. §reklama
  ниже). _Ранние прогоны PoC 2026-06-08 ошибочно дропали продажи внутри темы
  `reklama` — это исправлено этим правилом._ Объявления о пропаже/находке/пристрое
  животных — соцобъявление (`keep`) в любой теме.
- **Паблик `9363816`** («Типичный Киров»-style) постит областной
  развлекательно-погодный контент без местного хука (любительские прогнозы
  погоды, «первые маслята», календарные «дни друзей/…» с фото-стоков). →
  `drop` `развлекательный-репост` / `не-район` — если нет малмыжского повода или
  практической пользы с местной привязкой.
- **Тема `reklama` — реклама в ней штатна, не повод для `drop`.** В этой рубрике
  сводки объявления барахолки (вахта, услуги, продажи, классифайды)
  публикуются намеренно → по умолчанию `keep`. `drop` внутри `reklama` — только
  для **федерального/не-районного** заброса (напр. мульти-региональный набор по
  контракту «из всех регионов России» с нацканалами), а не для обычного местного
  объявления.
- **Федеральные промо-репосты, которые кросс-постят несколько пабликов** (напр.
  навигатор «Будьучителем.рф», всероссийские акции-флешмобы ко Дню России
  `#ЗдесьМояРоссия`/Росдетцентр): один и тот же текст всплывает у `179280169`,
  `86517261`, у соседних районов и т.п. В НЕ-`neighbors` темах (`novost` и пр.) →
  `drop` `федеральный` **даже если издатель — местный паблик** (релевантность
  оценивается по содержанию, а не по тому, кто запостил). PoC-наблюдение
  2026-06-17. В теме `neighbors` сводки-сводки соседей оставляем (`keep`) —
  там задача в трансляции активности соседнего района.
- **Паблик `195033391` (Центр Кино, `center-kino.ru`)** постит киноафишу.
  Анонсы репертуара («фильм … уже в кино», расписание/билеты, Пушкинская карта) —
  `keep` (культурный репертуар для жителей). Но **рекламно-маркетинговые
  конкурсы-розыгрыши брендовых призов** (напр. «Кинокласс и Афанасий», розыгрыш
  шопперов/шляп за фото) — `drop` `реклама`.
- **Соседние районы (Уржумский, Омутнинский, Лебяжский, Немский и др.) в
  НЕ-`neighbors` темах** (`sport`/`novost`/`kultura`/`addons`) без малмыжского
  хука → `drop` `не-район` (напр. событие в Уржумской школе №2 в теме `sport`).
  В теме `neighbors` — штатно `keep`. Исключение: пропажа/находка/пристрой
  животных — соцобъявление (`keep`) в любой теме, **если** это локальный/доступный
  жителю случай (далёкая областная служебная собака — серая зона, по умолчанию
  `keep` по явному правилу о пристрое).
