# Визуальный редактор в Cursor

## Как открыть визуальный редактор сайта в Cursor

Cursor имеет встроенную функцию для визуального просмотра и редактирования веб-сайтов.

### Способ 1: Simple Browser (Простой браузер)

1. Откройте HTML файл (например, `/home/valstan/SETKA/web/templates/index.html`)
2. Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac)
3. Введите: `Simple Browser: Show`
4. Введите URL: `https://3931b3fe50ab.vps.myjino.ru/`
5. Сайт откроется в встроенном браузере Cursor

### Способ 2: Live Preview (Живой предпросмотр)

1. Установите расширение "Live Preview" (если не установлено)
2. Откройте HTML файл
3. Нажмите правой кнопкой мыши на файл
4. Выберите "Show Preview" или "Open with Live Server"
5. Сайт откроется в предпросмотре с возможностью редактирования

### Способ 3: Команда из командной палитры

1. Нажмите `Ctrl+Shift+P`
2. Введите: `Live Preview: Show Preview`
3. Выберите файл для предпросмотра

### Способ 4: Горячие клавиши

- `Alt+L` - открыть Live Preview
- `Ctrl+Shift+V` - предпросмотр Markdown (для .md файлов)

## Настройка для проекта SETKA

Файл `.vscode/settings.json` уже настроен для работы с визуальным редактором.

### Открыть сайт напрямую:

1. Нажмите `Ctrl+Shift+P`
2. Введите: `Simple Browser: Show`
3. Введите URL: `https://3931b3fe50ab.vps.myjino.ru/`

### Редактирование в реальном времени:

1. Откройте файл шаблона (например, `web/templates/index.html`)
2. Нажмите правой кнопкой → "Show Preview"
3. Изменения в коде будут отображаться в предпросмотре

## Полезные расширения

Для лучшей работы с визуальным редактированием установите:

- **Live Preview** - встроенный предпросмотр
- **Live Server** - локальный сервер для разработки
- **HTML Preview** - предпросмотр HTML файлов

## Быстрый доступ

Создайте файл `.vscode/tasks.json` для быстрого запуска:

```json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "Open Site in Browser",
            "type": "shell",
            "command": "cursor --open-url https://3931b3fe50ab.vps.myjino.ru/",
            "problemMatcher": []
        }
    ]
}
```

Затем используйте `Ctrl+Shift+P` → `Tasks: Run Task` → `Open Site in Browser`

---

**Примечание**: Визуальный редактор в Cursor позволяет просматривать сайт и видеть изменения в реальном времени, но для полноценного визуального редактирования (drag-and-drop) используйте специализированные инструменты.
