# Снапшот выученных правил классификатора (ADR-0005)

2. [id=5 routine, утв. 2026-07-09] Посты от Клуба Малмыжских Путешественников (экскурсии, туры, мероприятия, ID паблика 226176537) → action=publish, theme='Клуб Малмыжских Путешественников'. Локальная туристическая организация, интересна жителям района.
3. [id=6 routine, утв. 2026-07-09] Посты от ВУЗов/колледжей об информации для поступления, днях открытых дверей, образовательных грантах, информационные письма для учеников → action=publish, theme='Образование' (не reklama). Newsworthy контент для молодёжи района.
4. [id=7 routine, утв. 2026-07-10] Контент о религиозных событиях, праздниках, церковных мероприятиях (концерты, богослужения, духовные праздники) → theme «Православие», action publish.
5. [id=8 routine, утв. 2026-07-10] Посты от детских садов, детских организаций про мероприятия, обучение в группе → theme «Дошкольное образование», action publish.
6. [id=12 routine, утв. 2026-07-12] Туристические маршруты, экскурсии, достопримечательности и природные объекты района → action=publish
7. [id=13 routine, утв. 2026-07-13] Подозрительные схемы удаленной работы (вакансии с неясными обязанностями, 'общение в чатах', 'переписка' и обещанием оклада) → delete как спам и потенциальные аферы.
8. [id=18 routine, утв. 2026-07-14] Пост без текста: если во вложениях есть читаемые фото или PDF — классифицировать по их содержимому (заполнить media_summary), не удалять автоматически. Если вложения нераспознаваемы (только видео/аудио, либо картинки/PDF не открылись) → action=delete.
9. [id=19 routine, утв. 2026-07-14] Частные объявления куплю/продам — по географии: от жителя СВОЕГО района (Малмыж, сёла района) → publish; из ЧУЖОГО района/города (Нолинск, Уржум и т.п., в т.ч. посты нолинских/уржумских пабликов) → delete. Перекупщики и скупка («срочный выкуп авто», «куплю авто в любом состоянии», «скупка битых») → delete всегда, независимо от географии.

12. [id=22 routine, утв. 2026-07-14] Местные вакансии с конкретным работодателем и местом работы в районе («требуется грузчик, Малмыж, ул. …», вакансии администрации/учреждений) → publish. Официальные опросы и голосования гос/муниципальных учреждений (forms.mkrf.ru, Росмолодёжь, оценка работы библиотек/ДК) → publish. Не путать с расплывчатыми онлайн-«вакансиями» без работодателя — те по-прежнему delete.
